ScanGo 태블릿 설치

새 태블릿은 QR 한 번으로 키오스크까지 자동 설정됩니다. 이미 쓰던 태블릿은 아래에서 앱만 내려받아 설치하세요.

새 태블릿 · PC 불필요

1. QR 스캔으로 자동 설치

공장초기화 상태에서 QR을 스캔하면 앱 설치 + 키오스크 잠금(device owner)까지 한 번에 끝납니다.

  1. 태블릿을 공장초기화 합니다 (설정 → 시스템 → 초기화).
  2. 첫 환영 화면(“Hi there”)을 손가락으로 6번 연속 탭 합니다.
  3. QR 스캐너가 열리면 WiFi를 선택해 연결 합니다.
  4. 아래 QR을 스캔합니다. 앱이 자동으로 다운로드·설치되고 키오스크로 잠깁니다.
  5. 설치 후 Account ID · License Code 를 입력해 활성화합니다.
  6. (권장) 설정 → 디스플레이 → 탐색 모드 → 제스처 탐색 으로 바꿉니다.
    키오스크 잠금에서도 Android 는 하단 뒤로 버튼만은 남깁니다(홈·최근은 사라집니다). 눌러도 아무 일이 없고 앱이 1초 안에 도로 숨기지만, 제스처 탐색으로 바꾸면 아예 보이지 않습니다.
스캔 대상: ScanGo 2.4.4 (244) · 릴리즈할 때마다 자동 갱신됩니다
활성화가 끝나면 아래 3번 절차로 원격지원을 켜 주세요. 버튼 두 번이면 끝납니다.
기존 태블릿 · 앱만 설치

2. 브라우저로 직접 설치

태블릿 브라우저로 이 페이지를 열고 아래 버튼을 누르면 됩니다. 최초 1회 “알 수 없는 앱 설치 허용”이 필요합니다.

ScanGo 앱 내려받기ScanGo 2.4.4 (244) · 설치 후 자동 업데이트됨↓ 릴리즈 노트 보기버전별 변경 내용→
이미 설치된 태블릿은 직접 받을 필요가 없습니다. 앱이 부팅 시와 6시간마다 스스로 최신 버전을 확인해 자동 설치합니다.
이 방법으로 설치하면 접근성을 켤 때 “제한된 설정(Restricted setting)” 이라며 막힙니다. Android 13+ 가 브라우저로 받은 앱에 거는 보안 제한입니다. 아래처럼 한 번 풀어 주세요.
설정 → 앱 → 전체 앱 보기 → ScanGo → 우측 상단 ⋮ → 제한된 설정 허용
⋮ 에 그 항목이 안 보이면, 접근성에서 ScanGo 켜기를 한 번 시도해 그 팝업을 띄운 뒤 다시 열면 나타납니다. 1번(QR 설치) 로 설치한 태블릿은 이 제한이 걸리지 않습니다.
PC 불필요

3. 원격지원 켜기

지원팀이 태블릿 화면을 보고 조작할 수 있게 합니다. 앱이 알아서 합니다 — RustDesk를 따로 내려받거나, 암호나 접속번호를 손으로 입력할 일이 없습니다.

  1. ScanGo에서 ☰ → 직원 PIN → SETTINGS → REMOTE 로 들어갑니다.
  2. ScanGo accessibility 줄의 TURN ON 을 누릅니다. 접근성 목록이 열리면 ScanGo → 스위치 켜기 → Allow. 끝나면 태블릿이 알아서 ScanGo로 돌아옵니다.
    이 한 단계만 사람이 해야 합니다. 접근성 켜기는 Android 보안 정책상 앱이 대신 할 수 없습니다. 이미 켜져 있으면 ON 으로만 표시됩니다.
  3. RustDesk auto setup 줄의 RUN 을 누릅니다.
    약 2분 걸립니다. 화면이 RustDesk와 설정 화면을 저절로 오갑니다 — 끝날 때까지 태블릿을 만지지 마세요. 끝나면 ScanGo 화면으로 돌아옵니다.
  4. 같은 REMOTE 화면의 Remote support ID 에 접속번호가 표시되면 완료입니다.
RUN 이 자동으로 처리하는 것
  • RustDesk 설치 (없을 때만)
  • 필요한 권한 승인 · 화면공유 시작
  • 영구 암호 설정 — 모든 매장 태블릿 공통, 값은 내부 운영 가이드(OPERATIONS-ko.md) 참고 (공개 페이지라 여기에는 적지 않습니다)
  • 암호로만 접속 허용 · 일회용 암호 끄기
  • 원격 조작(Input control) 켜기
  • 매장 화면을 가리는 RustDesk 아이콘 숨기기
  • 접속번호를 읽어 앱에 저장
접속번호는 손님 화면에서도 확인할 수 있습니다 — 우측 상단 모서리를 5번 연속 탭 하면 원격지원 팝업이 열립니다.
재부팅해도 원격지원은 저절로 살아납니다. 앱이 부팅 후 화면공유를 다시 켭니다(약 1분). 그래도 화면이 안 나오면 RUN 을 한 번 더 누르면 됩니다.

4. 라이선스가 만료·정지되면

앱은 시작할 때마다 라이선스 상태를 확인합니다.

  1. 만료(EXPIRED)·정지(SUSPENDED) 상태면 빨간 경고 화면이 뜨고 사용이 막힙니다.
  2. 포털에서 상태를 해제한 뒤 CHECK AGAIN 을 누르면 바로 복구됩니다.
  3. 다른 라이선스 코드로 다시 등록하려면 RE-ACTIVATE → 활성화 화면에서 새 코드를 입력합니다.
인터넷이 끊겨 상태를 확인하지 못할 때는 막지 않습니다 — 통신 장애로 매장 영업이 멈추지 않도록 한 설계입니다.
