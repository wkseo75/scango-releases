ScanGo 태블릿 설치

새 태블릿은 QR 한 번으로 키오스크까지 자동 설정됩니다. 이미 쓰던 태블릿은 아래에서 앱만 내려받아 설치하세요.

새 태블릿 · PC 불필요

1. QR 스캔으로 자동 설치

공장초기화 상태에서 QR을 스캔하면 앱 설치 + 키오스크 잠금(device owner)까지 한 번에 끝납니다.

  1. 태블릿을 공장초기화 합니다 (설정 → 시스템 → 초기화).
  2. 첫 환영 화면(“Hi there”)을 손가락으로 6번 연속 탭 합니다.
  3. QR 스캐너가 열리면 WiFi를 선택해 연결 합니다.
  4. 아래 QR을 스캔합니다. 앱이 자동으로 다운로드·설치되고 키오스크로 잠깁니다.
  5. 설치 후 Account ID · License Code 를 입력해 활성화합니다.
  6. (권장) 하단 뒤로/홈 버튼을 없애려면 설정 → 디스플레이 → 탐색 모드 → 제스처 탐색 으로 바꿉니다. 앱에서는 바꿀 수 없는 시스템 설정입니다.
스캔 대상: ScanGo 2.3.3 (233) · 릴리즈할 때마다 자동 갱신됩니다
원격지원(RustDesk)은 QR로 설치되지 않습니다. 아래 3번 절차로 태블릿에서 바로 설치하세요.
기존 태블릿 · 앱만 설치

2. 브라우저로 직접 설치

태블릿 브라우저로 이 페이지를 열고 아래 버튼을 누르면 됩니다. 최초 1회 “알 수 없는 앱 설치 허용”이 필요합니다.

ScanGo 앱 내려받기ScanGo 2.3.3 (233) · 설치 후 자동 업데이트됨↓ 릴리즈 노트 보기버전별 변경 내용→
이미 설치된 태블릿은 직접 받을 필요가 없습니다. 앱이 부팅 시와 6시간마다 스스로 최신 버전을 확인해 자동 설치합니다.
PC 불필요

3. 원격지원(RustDesk) 설치

지원팀이 태블릿 화면을 보고 도울 수 있게 합니다. 태블릿에서만 진행하며 PC가 필요 없습니다.

RustDesk 내려받기1.2.7 고정 · 최신 버전으로 업데이트하지 마세요↓
  1. ScanGo에서 ☰ → 직원 PIN → EXIT → PIN 재확인 으로 키오스크를 잠시 해제합니다. (EXIT 는 오너/어드민 권한 직원 PIN 으로만 가능)
  2. 위 버튼으로 RustDesk를 내려받아 설치합니다.
  3. RustDesk를 열고 경고창이 뜨면 "다시 표시 안 함" 체크 → I Agree, 알림 요청은 Allow.
  4. Share screen 탭 → Start service → 화면 녹화 요청에 Start now.
  5. 우측 상단 ⋮ → Set permanent password 로 영구 암호를 설정합니다. 모든 매장 태블릿에 같은 값을 쓰며, 값은 내부 운영 가이드(OPERATIONS-ko.md)를 따릅니다. 공개 페이지라 여기에는 적지 않습니다.
  6. 같은 화면의 ID(예: 1 228 570 464)를 확인합니다. 옆의 복사 아이콘을 쓰면 오타를 막을 수 있습니다.
  7. 설정 → 접근성 → RustDesk Input 과 ScanGo 를 모두 켭니다. (ScanGo 접근성은 재부팅 후 화면공유를 자동 승인하는 데 필요합니다.)
  8. ScanGo를 다시 열고 ☰ → 직원 PIN → SETTINGS → REMOTE 에서 6번의 ID를 입력합니다.
설정이 끝나면 태블릿에서 RustDesk 앱을 다시 열지 마세요 — 화면공유 세션이 끊깁니다. 끊겼다면 RustDesk를 열어 Stop service → Start service 순서로 다시 시작하세요 (이미 실행 중으로 보여도 화면이 안 나오면 반드시 한 번 멈췄다 켜야 복구됩니다).
참고

4. 라이선스가 만료·정지되면

앱은 시작할 때마다 라이선스 상태를 확인합니다.

  1. 만료(EXPIRED)·정지(SUSPENDED) 상태면 빨간 경고 화면이 뜨고 사용이 막힙니다.
  2. 포털에서 상태를 해제한 뒤 CHECK AGAIN 을 누르면 바로 복구됩니다.
  3. 다른 라이선스 코드로 다시 등록하려면 RE-ACTIVATE → 활성화 화면에서 새 코드를 입력합니다.
인터넷이 끊겨 상태를 확인하지 못할 때는 막지 않습니다 — 통신 장애로 매장 영업이 멈추지 않도록 한 설계입니다.
